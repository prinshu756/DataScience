## ✨ Overview
This repository is a simple but professional collection of data science notebooks and datasets that showcase practical Python-based analysis, visualization, and machine learning workflows.

It is designed to help you present your work confidently in an internship, college project, or portfolio setting.

## 📚 What’s inside
- [iris.ipynb](iris.ipynb) — Iris flower classification using a Random Forest model
- [unemployement.ipynb](unemployement.ipynb) — Advertising and sales prediction using linear regression
- CSV datasets for real-world experimentation and analysis

## 🚀 Highlights
- Data cleaning and preparation
- Exploratory data analysis (EDA)
- Visualizations with Matplotlib and Seaborn
- Machine learning model training and evaluation
- Internship-friendly project presentation flow

## 🗂️ Repository structure
```text
DataScience/
├── iris.ipynb

├── Advertising.csv

├── Unemployment in India.csv
└── Unemployment_Rate_upto_11_2020.csv
```

## ▶️ How to run
1. Open the notebooks in Jupyter Notebook or VS Code.
2. Install the required libraries:
   ```bash
   pip install pandas numpy matplotlib seaborn scikit-learn jupyter
   ```
3. Run the cells in order to reproduce the analysis and visualizations.
